# IIITH_Internship_2026 – Computer Vision & YOLO Experiments

This repository contains weekly work completed during my internship at **IIIT Hyderabad**.
The internship focuses on video processing, object detection, semantic segmentation, and creating custom datasets using YOLO.

The work is organized week-by-week, showing the complete pipeline from **video → frames → detection → segmentation → custom dataset creation**.

---

## Internship Objectives

* Learn video processing using **FFmpeg**
* Work with **Python virtual environments**
* Perform **Object Detection using YOLO**
* Perform **Semantic Segmentation**
* Understand YOLO dataset structure & metrics
* Create and label a **custom dataset**
* Maintain reproducible research using GitHub

---

## Repository Structure

```
Week1 – Video Processing using FFmpeg  
Week2 – YOLO Object Detection  
Week3 – Semantic Segmentation & Metrics  
Week4 – YOLO Dataset & Labeling
```

Each folder contains:

* Task description
* Commands/scripts used
* Screenshots & results
* Generated outputs

---

## Workflow Covered in This Internship

1. Download video → extract frames
2. Reconstruct video from frames
3. Add background music
4. Run YOLO object detection on frames
5. Convert detections back to video
6. Perform semantic segmentation
7. Stack videos for comparison
8. Study YOLO configuration & metrics
9. Create labeled dataset using Label Studio
